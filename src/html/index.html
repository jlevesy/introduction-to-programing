Program ? Code ? Instructions ?

What does it means ?

What is a program ?

A series of instructions a system can execute

In order to produce something usefull or not...

Expressed in a language both the computer and the human can understand !

A parallel with cooking...

Imagine a cooking recipe

It's composed of instructions !

Like: "mix eggs with flour and sugar in a bowl"

The recipe is the program

And you're the system executing it ;)

Instructions for computers !

Assign value 10 to variable named 'age'

If 'age' is lower than 6 then call function 'greetChildren'

Which translates in javascript into... 


              var age = 10; // Assign value 10 to variable age
              if (age < 6) { // If age is lower than 6
                greetChildren(); // call function greetChildren
              }
